    • About Me

    דוד בן גוריון היה ראש הממשלה הראשון של מדינת ישראל. הוא נולד ב-1 בדצמבר 1973 בשם דוד יוסף גרין לשיינדל ואביגדור גרין בעיירה פלונסק שבפולין, במשפחה היו חמישה ילדים – שלושה בנים ושתי בנות. דוד היה הבן הרביעי.
    בן־גוריון היה איש העלייה השנייה, ממנהיגי תנועת העבודה, חבר במפלגות פועלי ציון ואחדות העבודה, מזכירה הכללי הראשון של ההסתדרות הכללית של העובדים בארץ ישראל ומנהיגה הראשון של מפא"י.
    בתפקידיו כיושב ראש הנהלת הסוכנות היהודית ויושב ראש ההנהלה הציונית, עמד בראש הנהגת היישוב העברי בארץ ישראל בתקופת המנדט הבריטי, ובתפקידו כיושב ראש מנהלת העם מילא תפקיד מרכזי בניסוח מגילת העצמאות, והכריז ב־14 במאי 1948 על הקמת מדינת ישראל. עם הקמת המדינה החל לפעול למען הממלכתיות, ובזמן מלחמת העצמאות החליט על פירוק המחתרות, וחתם על הפקודה להקמת צה"ל. על מנת לקיים צבא ממלכתי אחד, הורה להפגיז את האונייה אלטלנה ופירק את מטה הפלמ"ח.

    Q&A

    Q1: Main differences between Web 1.0, Web 2.0 and Web 3.0

    Web 1.0

    1. "read" web(read only).
    2. Millions of users.
    3. Static pages - users can only read.
    4. A network that presents content and information to the user.


    Web 2.0

    1. "read-write" web.
    2. Billions of users.
    3. The user can interact with the website(not read only), two way communication and collaborative approach.
    4. Connect people by social networks.


    Web 3.0

    1. "read-write-execute" web.
    2. Trillions of users.
    3. Personalaized experience by using AI - the web can learn the user for relevant reccomendations and info.
    4. The data stored on distributed nodes, the users not need to think about the suspension of a particular account or service disruptions due to technical or other reasons.
    5. The web development includes "backend side" and database.
